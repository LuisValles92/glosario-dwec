eventos_24 - Arrastrar y soltar: Mover al cuadro azul el texto y la imagen

* target: El objeto donde se produjo el evento, el objeto origen.

* currentTarget: El objeto que está gestionando/manejando el evento.

FASE 1 + Es - ondragstart: Ocurre cuando el usuario comienza a arrastrar un elemento.
e.dataTransfer.setData("", e.target.id);
(Ver id del elemento target en consola)

FASE 2 + CA - ondragover: Ocurre cuando un elemento que se puede arrastrar se arrastra sobre un destino de colocación válido.
e.preventDefault();
(Ver id del elemento target en consola)

FASE 3 + CA - ondrop: Ocurre cuando un elemento arrastrable se coloca en un destino de colocación válido. Aparece el cuadradito de que se puede depositar.
e.preventDefault();
var datos = e.dataTransfer.getData("");
e.target.appendChild(document.getElementById(datos));
(Ver id del elemento target en consola)
